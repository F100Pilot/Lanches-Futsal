⚙️ Configuração da API

Cola abaixo a URL do teu Google Apps Script. Se ainda não tens, segue os passos indicados.

Como obter a URL:
  1. Cria uma Google Sheet com colunas: ID, Nome do Atleta, Data, Entregue, Observações
  2. Vai a Extensions → Apps Script
  3. Cola o código do ficheiro Code.gs (incluído no projeto)
  4. Clica em Deploy → New deployment → Web app
  5. Execute as: Me · Who has access: Anyone
  6. Autoriza e copia a URL gerada

🔐 Password: Protege os settings (roda dentada).
🔑 Chave de Acesso: Apenas tu consegues adicionar/editar/deletar registos.
